What I Learned
CareerPath AI Chatbot
Building AI Tools Requires Empathy First
CareerPath AI showed me that great AI products start with understanding people — not technology. Using the Design Thinking framework forced me to define real user problems before writing a single prompt.
💡
Key lesson: Prompt quality drives output quality. Vague prompts produce vague answers. Specific, context-rich prompts unlock the real power of LLMs.
AI Lab Experiments
Studying AI Is as Valuable as Using It
The lab experiments revealed that AI tools behave very differently depending on how you talk to them. Systematically testing and comparing outputs is a skill that separates practitioners from casual users.
🔍
Key lesson: Context changes everything. Adding personal background to prompts produced significantly more relevant and actionable AI responses.
